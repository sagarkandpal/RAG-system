# RAG Application 🚀

This project is a Retrieval-Augmented Generation (RAG) application built using LangChain, Vector Databases, Embeddings, and LLMs.

The system retrieves relevant information from documents and generates accurate AI-powered responses using Large Language Models.

---

# 📦 Features

- Document Loading
- Text Chunking
- Embedding Generation
- Vector Database Storage
- Semantic Search
- Retrieval-Based Question Answering
- Chat with PDFs/Documents
- LangChain Integration

---

# 🛠 Tech Stack

- Python
- LangChain
- FAISS / ChromaDB
- OpenAI / Groq / Mistral
- HuggingFace Embeddings
- Streamlit
